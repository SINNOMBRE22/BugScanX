## 🎯 Resumen

**BugScanX** es una herramienta avanzada de descubrimiento de *bug hosts* diseñada para encontrar hosts SNI y HTTP funcionales, ideales para aplicaciones de túnel y VPN. Esta versión incluye ajustes específicos en la interfaz, traducciones al español y optimizaciones de rendimiento para VPS y Termux.

---

## ✨ Características Principales

### 🎯 Host Scanner
Escaneo multi-modo de alto rendimiento:
- **Direct Scanning** — Descubrimiento de bugs HTTP/HTTPS con métodos personalizados.
- **DirectNon302** — Excluye automáticamente redirecciones para encontrar bugs reales.
- **SSL/SNI Analysis** — Análisis profundo de certificados para hosts SNI.
- **Multi-threading** — Procesamiento concurrente masivo para escaneos rápidos.

### 🔍 Enumeración de Subdominios
Descubrimiento profesional de subdominios utilizando múltiples proveedores de API y motores de búsqueda para expandir tus listas de objetivos.

### 🌐 Inteligencia IP y Redes
- **Reverse IP Lookup** — Revela todos los dominios alojados en una dirección IP.
- **Procesamiento CIDR** — Conversión y análisis masivo de rangos de red.
- **DNS Analysis** — Validación completa de registros DNS y seguridad SSL.

### 📁 File Toolkit
Herramientas integradas para procesar listas de hosts: dividir archivos grandes, unir partes, eliminar duplicados y filtrar por palabras clave o extensiones TLD.
